# 오늘 뭐 먹지? · MUKVELET

내 주변 맛집을 예산·취향에 맞춰 추천해 주는 웹앱. 전국 어디서나 `현재 위치`로 동작합니다.

**➡️ 배포 주소:** `https://<사용자이름>.github.io`

## 기능
- 시/도·시군구 선택 또는 **현재 위치**(GPS) 기반 추천
- 반경 / 예산(예상가) / 먹고 싶은 종류 필터
- 최근 먹은 메뉴 종류·가게 제외
- 요청 개수를 채울 때까지 반경 자동 확장
- 카카오·네이버 지도 링크, 지도(OpenStreetMap) 표시

## 데이터
소상공인시장진흥공단 상가(상권)정보(공공데이터) 기반 전국 음식점.
가격은 업종 기준 **추정치**이며 실제와 다를 수 있습니다.

## 구성
- `index.html` — 앱 전체(HTML/CSS/JS)
- `data.js` — 음식점 데이터 (`build_data.py --all` 로 생성)
